Given the `head` of a singly linked list, return `true` _if it is a_

_palindrome_

_or_ `false` _otherwise_.

**Example 1:**

![img.png](img.png)

```
Input: head = [1,2,2,1]
Output: true

```

**Example 2:**

![img_1.png](img_1.png)

```
Input: head = [1,2]
Output: false

```

**Constraints:**

-   The number of nodes in the list is in the range `[1, 10^5]`.
-   `0 <= Node.val <= 9`

**Follow up:** Could you do it in `O(n)` time and `O(1)` space?
